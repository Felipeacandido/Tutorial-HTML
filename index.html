Exemplos básicos de HTML

Neste capítulo mostraremos alguns exemplos básicos de HTML. Não se preocupe se usarmos tags que você ainda não conhece.

Documentos HTML

Todos os documentos HTML devem começar com uma declaração de tipo de documento: <!DOCTYPE html>. O próprio documento HTML começa <html> e termina com </html>. A parte visível do documento HTML está entre <body> e </body>.

Exemplo

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading<h1>
<p>My first paragraph.<p>

</body>
</html>

A declaração <!DOCTYPE>

A <!DOCTYPE> declaração representa o tipo de documento e ajuda os navegadores a exibir as páginas da web corretamente. Deve aparecer apenas uma vez, no topo da página (antes de qualquer tag HTML). A <!DOCTYPE> declaração não diferencia maiúsculas de minúsculas. A <!DOCTYPE> declaração para HTML5 é:

<!DOCTYPE html>

<h1> define o título mais importante. <h6> define o título menos importante:

Exemplo

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>

Elementos

Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final.

Elementos HTML

O elemento HTML é tudo, desde a tag inicial até a tag final:
<tagname> O conteúdo vai aqui... </tagname>
Exemplos dealguns elementos HTML:
<h1> Meu primeiro título </h1>
<p> Meu primeiro parágrafo.</p>

Elementos HTML animados

Os elementos HTML podem ser animados (isso significa que os elementospodem conter outros elementos).
Todos os documentos HTML consistem em elementos HTML animados.
O exemplo a seguir contém quatro elementos HTML (,<html> e ) :<body> <h1> <p>

Exemplo

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading<h1>
<p>My first paragraph.<p>

</body>
</html>

Ecemplo explicado

O <html> elemento é o elemento raiz e define todo o documento HTML.
Possui uma tag inicial <html> e uma tag final </html>.
então, dentro do <html> elemento existe um <body> elemento:

<body>

<h1>My First Heading<h1>
<p>My first paragraph.<p>

</body>

O <body>elemento define o corpo do documento.
Possui uma tag inicial <body>e uma tag final </body>.
Então, dentro do <body>elemento existem outros dois elementos: <h1>e <p>:

<h1>My First Heading</h1>
<p>My first paragraph.</p>

O <h1> elemento define um título.
Possui uma tag inicial <h1> e uma tag final </h1>:

<h1>My first Heading.</h1>

O <p> elemento define um parágrafo.
Possui uma tag inicial <p> e uma tag final </p>:

<p> My first paragraph.</p>

Nunca pule a tag final

Alguns elementos HTML serão exibidos corretamente, mesmo se você esquecer a tag final:

Exemplo

<html>
<body>

<p>This is a paragraph.<p>
<p>This is a paragraph.<p>

</body>
</html>

No entanto, nunca confie nisso! Resultados inesperados e erros podem ocorrer se você esquecer a tag final!

Elementos HTML vazios

Elementos HTML sem conteúdo são chamados de elementos vazios. A <br>tag define uma quebra de linha e é um elemento vazio sem tag de fechamento:

Exemplo

<p>This is a <br> paragraph with a line break.</p>

HTML não diferencia maiúsculas de minúsculas

As tags HTML não diferenciam maiúsculas de minúsculas: <P>significa o mesmo que <p>


O padrão HTML não exige tags em minúsculas.

Atributos HTML

  • Todos os elementos HTML podem ter atributos
  • Os atributos fornecem informações adicionais sobre os elementos
  • Os atributos são sempre especificados na tag inicial
  • Os atributos geralmente vêm em pares nome/valor como: nome="valor"

    O atributo src

    A <img>tag é usada para incorporar uma imagem em uma página HTML. O srcatributo especifica o caminho para a imagem a ser exibida:

    Exemplo

    <img src="img_girl.jpg">

    Os atributos de largura e altura

    A <img>tag também deve conter os atributos width e height, que especificam a largura e a altura da imagem (em pixels):

    Exemplo

    <img src="img_girl.jpg" width="500" height="600">

    O atributo alternativo

    altO atributo obrigatório para a <img> tag especifica um texto alternativo para uma imagem, se a imagem por algum motivo não puder ser exibida. Isso pode ser devido a uma conexão lenta, a um erro no srcatributo ou ao usuário usar um leitor de tela.

    Exemplo

    <img src="img_girl.jpg" alt="Girl with a jacket">

    O atributo de estilo

    O styleatributo é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

    Exemplo

    <p style="color:red;">This is a red paragraph.

    Cabeçalhos

    Cabeçalhos HTML são elementos fundamentais para organizar e destacar títulos ou legendas em uma página da web, utilizando as tags de <h1> a <h6>. A tag <h1> é usada para o título mais significativo, enquanto <h6> é para o menos importante, seguindo uma hierarquia que ajuda tanto na estruturação do conteúdo quanto na indexação por mecanismos de busca. Essa hierarquia também facilita a navegação dos usuários, que muitas vezes escaneiam a página pelos títulos. Além disso, o tamanho dos cabeçalhos pode ser personalizado com CSS, especificamente através do atributo style utilizando a propriedade font-size, permitindo ajustes visuais conforme necessário para adequar a apresentação ao design da página.

    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>
    <h4>Heading 4</h4>
    <h5>Heading 5</h5>
    <h6>Heading 6</h6>

    Parágrafos HTML

    Parágrafos em HTML são definidos pela tag <p>, começam sempre em uma nova linha e são separados por margens automáticas adicionadas pelo navegador, que também ignora espaços e linhas extras dentro do elemento. Telas de diferentes tamanhos e janelas redimensionadas podem alterar a exibição do HTML, mas não é possível controlar isso apenas adicionando espaços ou linhas no código HTML.
    A tag <hr> é utilizada para inserir uma linha horizontal na página, funcionando como uma separação temática entre conteúdos. Ela é uma tag vazia, ou seja, não requer uma tag de fechamento.
    Para quebras de linha específicas sem iniciar um novo parágrafo, usa-se a tag <br>, outra tag vazia do HTML.
    Quando é necessário manter a formatação original do texto, como em poesias ou códigos, o elemento <pre> é apropriado, pois preserva espaços e quebras de linha e geralmente apresenta o texto em uma fonte de largura fixa, mantendo a disposição visual pretendida pelo autor.

    Exemplo

    <h1>Isso é o cabeçalho 1</h1>
    <p>Isso é um paragrafo.</p>
    <hr>
    <h2>Isso é o cabeçalho 2</h2>
    <p>Isso é outro paragrafo.</p>
    <hr>

    Estilos HTML

    O atributo style em HTML é usado para aplicar estilos diretamente a elementos individuais, permitindo ajustes como cor, tamanho da fonte, família da fonte, alinhamento do texto e cor de fundo. A sintaxe envolve style="property:value;", com "property" sendo uma propriedade CSS e "value" o valor correspondente a essa propriedade.
    Este atributo permite alterações rápidas e específicas na apresentação dos elementos da página, como modificar cores de fundo e de texto, escolher a família da fonte, ajustar o tamanho do texto, e definir o alinhamento do texto dentro dos elementos. É uma forma eficaz e imediata de controlar a estética de componentes específicos em uma página web.

    Exemplo

    <h1 style="color:blue;">This is a heading</h1>
    <p style="color:red;">This is a paragraph.</p>
    <h1 style="font-family:verdana;">This is a heading</h1>
    <p style="font-family:courier;">This is a paragraph.</p>

    Formatação

    Citações

    Este capítulo aborda os elementos HTML relacionados à formatação e citação de texto. O elemento <blockquote> é utilizado para destacar uma seção citada de outra fonte, comumente usado para citações longas, e os navegadores geralmente aplicam um recuo automático a esses elementos. O <q> é empregado para citações curtas, e os navegadores frequentemente inserem aspas ao redor do texto citado. Para abreviaturas e acrônimos, a tag <abbr> é utilizada, fornecendo informações úteis aos navegadores e motores de busca, sendo possível adicionar uma descrição ao passar o mouse sobre o elemento. O <address> é empregado para fornecer informações de contato do autor ou proprietário de um documento ou artigo, geralmente sendo renderizado em itálico e com uma quebra de linha antes e depois do elemento. A tag <cite> é utilizada para definir o título de um trabalho criativo, como um livro ou filme, e geralmente é renderizada em itálico. Por fim, <bdo> é usado para substituir a direção do texto, permitindo a escrita bidirecional, com a direção do texto especificada pelo atributo dir.

    Exemplo

    <p>A <abbr title="Organização Mundial de Saude">OMS</abbr> doi dundada em 1948.</p>
    <p>o objetivo das lutas do WWE é: <q>não sei prefiro MMA.</q></p>
    <p>a musica <cite>Like a Stone</cite> é uma das melhores musicas da banda Audioslave.</p>

    Comentários

    Os comentários HTML são utilizados para adicionar notas ou lembretes ao código-fonte HTML sem que sejam exibidos no navegador. Eles são inseridos entre <!-- e -->. Comentários podem ser úteis para documentar o código ou ocultar temporariamente partes do conteúdo. Por exemplo, você pode comentar uma seção do código para explicar sua função ou para desativar temporariamente uma parte do código sem removê-la completamente. Comentários também podem ser usados para ocultar parte do conteúdo dentro de um elemento específico, ajudando na compreensão do código para quem está trabalhando nele.

    Exemplo

    <!-- titulo -->
    <h1>A volta dos que não foram.</h1>
    <!-- texto -->
    <p>...? </p>

    Cores

    As cores em HTML podem ser especificadas de diversas maneiras: por nomes de cores predefinidos, valores RGB, valores HEX, valores HSL, valores RGBA ou valores HSLA. Os nomes de cores predefinidos incluem uma variedade de tons, como "Tomate", "Laranja", "Azul ardósia", entre outros, totalizando 140 nomes de cores padrão. Você pode definir a cor de fundo de elementos HTML usando o atributo style com a propriedade background-color. Por exemplo:

    <h1 style="background-color:DodgerBlue;">Hello World</h1>
    <p style="background-color:Tomato;">Lorem ipsum...</p>

    Para definir a cor do texto, utiliza-se a propriedade color no atributo style:

    <h1 style="color:Tomato;">Hello World</h1>
    <p style="color:DodgerBlue;">Lorem ipsum...</p>
    <p style="color:MediumSeaGreen;">Ut wisi enim...</p>

    A cor da borda de elementos pode ser definida com a propriedade border. Por exemplo:

    <h1 style="border:2px solid Tomato;">Hello World</h1>
    <h1 style="border:2px solid DodgerBlue;">Hello World</h1>
    <h1 style="border:2px solid Violet;">Hello World</h1>

    Além dos nomes de cores, é possível especificar cores utilizando valores RGB, HEX ou HSL. Por exemplo:

    <h1 style="background-color:rgb(255, 99, 71);">...<h1>
    <h1 style="background-color:#ff6347;">...</h1>
    <h1 style="background-color:hsl(9, 100%, 64%);">...v/h1>

    Também é possível adicionar transparência às cores usando os valores RGBA ou HSLA:

    <h1 style="background-color:rgba(255, 99, 71, 0.5);">...</h1>
    <h1 style="background-color:hsla(9, 100%, 64%, 0.5);">...</h1>

    Essas diferentes maneiras de especificar cores oferecem flexibilidade para o design e estilização de elementos HTML em uma página da web.

    Links

    Os links HTML são essenciais em páginas da web, permitindo a navegação entre diferentes páginas ou recursos online. Eles são criados com <a> e têm a estrutura <a href="url">texto do link</a>, onde o atributo href indica o destino. O atributo target pode ser usado para determinar onde abrir o link. Links podem ser absolutos (completos) ou relativos (referentes ao próprio site). Imagens podem ser usadas como links com <img> dentro de <a>. Para links de e-mail, use mailto: no href. Botões também podem agir como links com JavaScript. O atributo title fornece informações adicionais. Links absolutos vinculam a páginas externas, enquanto links relativos são para páginas do mesmo site. Esses são os fundamentos dos links em HTML.

    Exemplo

    <a href="https://www.w3schools.com/">Visit W3Schools.com!</a>
    <a href="default.asp">
    <img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">
    </a>

    Exemplo prático

    Visite o site da W3Schools.com para aprender mais!

Para adicionar um favicon ao seu site, salve sua imagem de favicon no diretório raiz do seu servidor web ou crie uma pasta no diretório raiz chamada imagens e salve sua imagem de favicon nesta pasta. Um nome comum para uma imagem de favicon é “favicon.ico”. Em seguida, adicione um <link>elemento ao seu arquivo “index.html”, após o <title>elemento, assim:

    </head>
    <body>

    Tabelas

    As tabelas HTML são usadas para organizar dados em linhas e colunas, e são compostas por células de tabela dentro de linhas e colunas. Cada célula é definida por `<td>` e `</td>`, onde `<td>` significa dados da tabela. Todo conteúdo entre ` <td>` e `</td>` é o conteúdo da célula. Cada linha começa com `<tr>` e termina com `</tr>`, onde `<tr>` significa linha da tabela. Você pode ter quantas linhas quiser, mas certifique-se de que o número de células seja o mesmo em cada linha. Se desejar que suas células sejam células de cabeçalho, use `<th>` em vez de ` <td>`.

    Exemplo

    <table>
    <tr>
    <th>Person 1</th>
    <th>Person 2</th>
    <th>Person 3</th>
    </tr>
    <tr>
    <td>Emil</td>
    <td>Tobias</td>
    <td>Linus</td>
    </tr>
    <tr>
    <td>16</td>
    <td>14</td>
    <td>10</td>
    </tr>
    </table>

    Block & line

    Elementos HTML são classificados em dois tipos principais de exibição: bloco e embutido.

    **Elementos de nível de bloco**: Estes elementos começam sempre em uma nova linha, ocupam toda a largura disponível e incluem uma margem automática. Exemplos comuns incluem `<p>` e `<div>`, ambos usados para definir áreas específicas como parágrafos e seções dentro de um documento HTML.

    **Elementos embutidos**: Eles não começam em uma nova linha e ocupam apenas a largura necessária para o conteúdo. Um exemplo é o `<span>`, um contêiner embutido usado para estilizar ou destacar partes do texto sem afetar a estrutura do documento.

    O `<div>` é amplamente usado para organizar outros elementos HTML e facilitar a estilização com CSS, enquanto o `<span>` é ideal para aplicar estilos a segmentos específicos de texto, também com o auxílio de CSS. Ambos os elementos são versáteis devido à ausência de atributos obrigatórios, mas frequentemente usam atributos como `style`, `class`, e `id` para personalização.

    Exemplo

    <p>Hello World</p>
    <div>Hello World</div>

    Divisão

    O ` <div>` no HTML é um contêiner genérico usado para agrupar e organizar seções de conteúdo e outros elementos dentro de um documento. Não tem significado semântico próprio, mas é fundamental na construção de layouts de páginas web, especialmente quando usado com CSS para aplicar estilos. Ele permite aos desenvolvedores criar layouts complexos e organizar o HTML de maneira clara, facilitando a manutenção e compreensão do código, especialmente em projetos grandes. O `<div>` também é útil para aplicar estilos específicos e para manipulação via JavaScript, tornando-o uma ferramenta versátil na web moderna.

    Classe

    As classes no HTML são atributos que podem ser adicionados a elementos para agrupá-los e estilizá-los de maneira semelhante. Elas permitem aplicar estilos CSS específicos a vários elementos com a mesma classe, tornando a estilização mais eficiente e consistente. Classes também são usadas para aplicar efeitos interativos e comportamentais usando JavaScript, permitindo alterar dinamicamente a aparência ou comportamento de elementos em resposta a ações do usuário. Em resumo, as classes são uma maneira eficiente e flexível de estilizar e interagir com o conteúdo da página HTML.

    Exemplo

    <h2 class="city main">London</h2> <h2 class="city">Paris</h2> <h2 class="city">Tokyo</h2>

    Id

    No HTML, o atributo "id" é usado para identificar exclusivamente um elemento específico em uma página. Ele permite selecionar e manipular esse elemento por meio de estilos CSS e scripts JavaScript. Cada elemento pode ter apenas um ID, e ele deve ser único dentro do documento HTML. Comparado ao atributo "class", que pode ser aplicado a vários elementos, o "id" é usado para identificar singularmente um elemento.

    Exemplo

    ```html
    <style>
    /* Estiliza o elemento com o id "myHeader" */
    #myHeader {
    background-color: lightblue;
    color: black;
    padding: 40px;
    text-align: center;
    }
    /* Estiliza todos os elementos com a classe "city" */
    .city {
    background-color: tomato;
    color: white;
    padding: 10px;
    }
    </style>

    <!-- Um elemento com um id único -->
    <h1 id="myHeader">My Cities</h1>

    <!-- Múltiplos elementos com a mesma classe -->
    <h2 class="city">London</h2>
    <p>London is the capital of England.</p>

    <h2 class="city">Paris</h2>
    <p>Paris is the capital of France.</p>

    <h2 class="city">Tokyo</h2>
    <p>Tokyo is the capital of Japan.</p>
    ```

    Iframes

    A tag `<iframe>` do HTML permite incorporar uma página da web dentro de outra página, exibindo o conteúdo de uma URL específica em uma área designada. Isso é útil para integrar conteúdo de terceiros, como vídeos do YouTube ou mapas do Google Maps, em um site. Embora seja conveniente, o uso de `<iframe>` pode apresentar desafios de segurança e deve ser usado com cautela.

    Exemplo

    <iframe src="url" title="description"></iframe> <iframe src="demo_iframe.htm" height="200" width="300" title="Iframe Example"></iframe>

    Codificação de URL

    A codificação de URL no HTML refere-se ao processo de transformar caracteres especiais e outros caracteres não permitidos em URLs em uma forma que possa ser transmitida e interpretada corretamente pelos navegadores da web. Isso é necessário porque certos caracteres, como espaços, acentos e símbolos especiais, não podem ser usados diretamente em uma URL.
    A codificação de URL é realizada substituindo cada caractere não permitido por sua representação em hexadecimal, precedida por um sinal de porcentagem (%). Por exemplo, um espaço é substituído por "%20".
    Este processo de codificação é comumente utilizado em links e parâmetros de consulta em URLs para garantir que elas sejam interpretadas corretamente pelos navegadores e servidores web. Por exemplo, ao enviar dados de um formulário através de uma solicitação GET, os valores dos campos são codificados para garantir que caracteres especiais, como espaços ou símbolos, sejam transmitidos corretamente.
    Em HTML, a codificação de URL também é aplicada automaticamente em certos elementos, como atributos href em links e valores de atributos src em elementos de mídia, quando os desenvolvedores utilizam caracteres especiais ou URLs contendo espaços ou símbolos. Isso garante que os URLs sejam interpretados corretamente pelos navegadores e servidores web.

    Exemplo

    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplos de Codificação de URL em HTML</title>
    </head>
    <body>

    <!-- Exemplo 1: Link com espaço no URL -->
    <a href="https://www.example.com/search?q=termo%20de%20busca">Link com espaço no URL</a>

    <br><br>

    <!-- Exemplo 2: Imagem com espaço no nome do arquivo -->
    <img src="https://www.example.com/imagem%20com%20espaço.jpg" alt="Imagem com espaço">

    <br><br>

    <!-- Exemplo 3: Formulário com campo de entrada contendo espaço -->
    <form action="https://www.example.com/processar-dados">
    <input type="text" name="nome" value="Jo%C3%A3o%20da%20Silva">
    <input type="submit" value="Enviar">
    </form>

    </body>
    </html>

    HTML vs XHTML

    XHTML, ou Extensible Hypertext Markup Language, é uma linguagem de marcação que segue as regras do XML (eXtensible Markup Language). Ela é uma versão mais rigorosa e estruturada do HTML, projetada para ser mais compatível com XML e mais precisa em relação à sintaxe.

    Principais características do XHTML:

    1. **Compatibilidade com XML:** O XHTML segue as regras do XML, o que significa que ele é mais estruturado e compatível com outras tecnologias baseadas em XML, como XSLT (Extensible Stylesheet Language Transformations) e XML Schema.

    3. **Separação de conteúdo e apresentação:** Assim como o HTML, o XHTML permite separar o conteúdo do documento da sua apresentação, usando folhas de estilo em cascata (CSS) para estilizar o conteúdo.

    4. **Suporte a namespaces:** O XHTML suporta namespaces XML, permitindo a inclusão de elementos e atributos de outras linguagens de marcação XML em documentos XHTML.

    5. **Compatibilidade com HTML:** O XHTML é compatível com as versões anteriores do HTML, o que significa que os documentos XHTML podem ser processados e exibidos por navegadores da web que suportam HTML.

    6. **Mais estruturação:** O XHTML promove uma estruturação mais clara e semântica do conteúdo da web, o que facilita a acessibilidade, a indexação por motores de busca e a manutenção do código.

    No entanto, o XHTML também é mais restrito do que o HTML em alguns aspectos, o que significa que pode ser mais difícil para os iniciantes.     Canvas

    O elemento <canvas> do HTML oferece uma área de desenho onde é possível criar gráficos, imagens e elementos visuais usando JavaScript. Ele possui uma API de desenho que permite manipular pixels, linhas, formas geométricas e texto. O canvas é ideal para criar gráficos interativos, animações e jogos, oferecendo alta performance e compatibilidade com os principais navegadores. É altamente personalizável e suporta uma variedade de técnicas de desenho, como gradientes, sombras e transformações. Exemplo de uso inclui desenhar formas, imagens e texto dentro do canvas, proporcionando uma experiência interativa aos usuários.

    Exemplo

              <canvas id="myCanvas" width="200" height="200" style="border:1px solid black;"></canvas>
          <script>
              // Obtém o elemento canvas
              var canvas = document.getElementById("myCanvas");
              // Obtém o contexto de renderização 2D
              var ctx = canvas.getContext("2d");
              // Define o centro do círculo
              var centerX = canvas.width / 2;
              var centerY = canvas.height / 2;
              // Define o raio do círculo
              var radius = 50;
              // Inicia o desenho do círculo
              ctx.beginPath();
              // Desenha o círculo
              ctx.arc(centerX, centerY, radius, 0, 2 * Math.PI);
              // Define a cor de preenchimento do círculo
              ctx.fillStyle = "red";
              // Preenche o círculo com a cor especificada
              ctx.fill();
              // Fecha o desenho do círculo
              ctx.closePath();
          </script>

    Arrastar/soltar

    O recurso de arrastar e soltar (drag and drop) em HTML permite que os usuários interajam com elementos da página movendo-os de um local para outro usando o mouse. É uma funcionalidade poderosa que pode ser implementada usando eventos JavaScript. Aqui está um resumo básico:
    Eventos Drag and Drop: Os principais eventos são dragstart, drag, dragenter, dragleave, dragover e drop. Elementos Principais: O elemento que pode ser arrastado deve ter o atributo draggable definido como true. O atributo ondragstart é usado para lidar com o início do evento de arrastar.
